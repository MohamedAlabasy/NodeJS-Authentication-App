## To run this project :   

`Step 1` :  Download the source code .
```
git https://github.com/MohamedAlabasy/NodeJS-View-Authentication.git
```

`Step 2` :  Enter the project file then install package
```
npm i
```
`Step 2` :  Enter the project file then ...
```
nodemon app.js 
```
or
```
npm run mon
```
